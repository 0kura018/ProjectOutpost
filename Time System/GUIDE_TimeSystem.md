# Система Тактов и Игрового Времени

## Краткое описание

Игра работает через **такты** — дискретные единицы времени. Время в игре идёт **только** когда ты тратишь такты. При трате тактов игра ускоряется, проходит игровое время, строительство продвигается.

---

## Основные понятия

| Понятие | Описание |
|---------|----------|
| **Смена** | 18 игровых часов (рабочий день) |
| **Такт** | Доля смены. Один такт = несколько игровых минут |
| **Игровой час** | 1 минута реального времени (при нормальной скорости) |
| **Ускорение** | При трате такта: 1 игровая минута = 5 реальных секунд |

---

## Формула тактов

```
Такты за смену = 10 базовых + (2 ? количество_крыс)
Максимум: 36 тактов
```

**Примеры:**
- 0 крыс ? 10 тактов
- 5 крыс ? 20 тактов
- 10 крыс ? 30 тактов
- 15+ крыс ? 36 тактов (максимум)

---

## Как пользоваться UI

### Ряд тактов

```
[?][?][?][?][?][?][?][?][?][?]
 ? потрачены    ? доступны
```

- **Зелёные** — доступные такты
- **Серые** — уже потраченные
- **Жёлтые** — подсвечены при наведении (будут потрачены)
- **Синие** — активно тратятся прямо сейчас

### Управление

| Действие | Результат |
|----------|-----------|
| **Навести** на такт | Подсветит этот такт и все после него |
| **ПКМ** на такт | Потратить этот такт и все после него |

**Пример:**  
Навёл на 5-й такт из 10 ? ПКМ ? потратятся такты 5, 6, 7, 8, 9, 10 (6 штук)

## Время и строительство

### Как связаны такты и время

```
Смена = 18 часов
30 тактов = 18 часов
1 такт = 18 / 30 = 0.6 часа = 36 минут
```

### BuildTime в RoomConfig

`BuildTime` теперь указывается в **игровых часах**, не в секундах!

```
BuildTime = 2.0  ?  Строится 2 игровых часа
                    ? 3-4 такта (при 30 тактах за смену)
```

### Прогресс строительства

- Строительство идёт **только** когда активен такт
- При трате тактов игра ускоряется, строительство продвигается
- Если не достроил за день — применяется **ночной штраф**

---

## Ночной штраф

Если строительство не завершено к концу смены:
- Прогресс откатывается на **1 игровой час**
- Можно отключить через исследование (TODO)

```csharp
// Отключить штраф
GameTimeManager.Instance.NightPenaltyEnabled = false;
```

## Настройка в Inspector

### GameTimeManager

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| Shift Duration Hours | 18 | Длина смены в игровых часах |
| Real Seconds Per Game Hour | 60 | 1 игровой час = X реальных секунд |
| Tick Speed Real Seconds | 5 | Ускорение: 1 игровая минута за X секунд |
| Base Ticks Per Shift | 10 | Базовые такты |
| Ticks Per Resident | 2 | Такты за жителя |
| Max Ticks Per Shift | 36 | Максимум тактов |
| Night Penalty Hours | 1 | Штраф в игровых часах |

### TickRowUI

| Параметр | Описание |
|----------|----------|
| Ticks Container | Transform куда спавнятся слоты тактов |
| Tick Prefab | Префаб одного слота (необязательно, создаст сам) |
| Tick Size | Размер слота в пикселях |
| Tick Spacing | Отступ между слотами |

## Использование из кода

### Потратить такты
```csharp
// Потратить 5 тактов
GameTimeManager.Instance.SpendTicks(5);

// Потратить все оставшиеся
GameTimeManager.Instance.SpendAllTicks();
```

### Добавить жителей
```csharp
GameTimeManager.Instance.AddResident(3);
GameTimeManager.Instance.SetResidentCount(10);
```

### Подписаться на события
```csharp
GameTimeManager.Instance.OnTickStarted += () => Debug.Log("Такт начался");
GameTimeManager.Instance.OnTickEnded += () => Debug.Log("Такт закончился");
GameTimeManager.Instance.OnShiftEnded += () => Debug.Log("Смена закончилась");
GameTimeManager.Instance.OnDayChanged += (day) => Debug.Log($"День {day}");
```

### Получить информацию
```csharp
int day = GameTimeManager.Instance.CurrentDay;
float time = GameTimeManager.Instance.CurrentTimeHours;
string formatted = GameTimeManager.Instance.GetFormattedTime(); // "14:30"
int ticksLeft = GameTimeManager.Instance.TicksRemaining;
int maxTicks = GameTimeManager.Instance.MaxTicksToday;
bool isActive = GameTimeManager.Instance.IsTickActive;
```

## Настройка сцены

1. Создай пустой GameObject `GameTimeManager`
2. Добавь компонент `TimeSystem.GameTimeManager`
3. Создай UI Canvas
4. Добавь Panel с `HorizontalLayoutGroup` для тактов
5. Добавь компонент `TimeSystem.UI.TickRowUI`
6. Назначь `Ticks Container` = Panel с LayoutGroup

### Минимальный UI:
```
Canvas
??? TimePanel
?   ??? DayText (Text)
?   ??? TimeText (Text)
?   ??? TicksContainer (Panel + HorizontalLayoutGroup)
?   ??? TickInfoText (Text) — подсказка при наведении
```

## Примеры сценариев

### Быстрое строительство
```
Комната: BuildTime = 1 час
Тактов: 30 (1 такт = 36 минут)
Нужно: ceil(60 / 36) = 2 такта
```

### Долгое строительство
```
Комната: BuildTime = 6 часов
Тактов: 30 (1 такт = 36 минут)
Нужно: ceil(360 / 36) = 10 тактов
```

### Оценка из кода
```csharp
Room room = ...;
int ticksNeeded = room.GetEstimatedTicksToComplete();
Debug.Log($"Осталось примерно {ticksNeeded} тактов");
```
